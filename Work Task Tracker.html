▶ How to get your cookie (click to expand)
  1. Open telushealth.atlassian.net in your browser (log in if asked)
  2. Press F12 to open DevTools → click the Network tab
  3. Click any request in the list (e.g. the first one)
  4. In the right panel scroll to Request Headers
  5. Find the cookie: row → right-click it → Copy value
  6. Paste below and click Update Cookie
